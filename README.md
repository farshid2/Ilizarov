Data related to modelling, simulation and analysis of Ilizarov external circular (ring) external fixators.
The file RingRadCompShellAbq.json has the results of simulation radial compression of Ilizarov full rings (with different holes diameters, ring diameters and number of holes) under a 1000 [N] load using ABAQUS software.
The data are for full Ilizarov rings with different hole diameters (3, 5, 7, 9 [mm]), ring diameters (80, 100, 110, 120, 130, 140, 150, 160, 180, 200, 220, 240 [mm]) and number of holes (0, 2, 4, 6, 8, ....), under a (radial) load of 1000 [N]. For example, if the data is read into a Python dictionary or a JavaScript object named RESULTS, the values of RESULTS["3"]["150"]["36"]["U1"] is the maximum radial deformation (in [m]) of a ring with 3 mm holes, (inner) diameter of 150 mm and 36 holes, while RESULTS["3"]["150"]["36"]["U2"], gives the maximum lateral deformation (in [m]) and RESULTS["3"]["150"]["36"]["SP"] is the maximum (von Mises) stress in Pascals [Pa].
